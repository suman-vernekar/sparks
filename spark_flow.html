How It Works:

1. Data Input: Weather data is loaded from a CSV file containing date, temperature, humidity, and wind speed.

2. Spark Session: Apache Spark distributed computing framework is initialized.

3. Data Transformation: Data is cleaned, date fields are parsed, and new columns are derived.

4. Aggregations: Min, max, and average temperatures are calculated across different time periods.

5. Results Output: Analysis results are displayed in tabular format with key insights.

# Sample PySpark Code Structure spark = SparkSession.builder.appName("WeatherAnalysis").getOrCreate() # Load data df = spark.read.csv("weather_data.csv", header=True, inferSchema=True) # Transform data df_cleaned = df.withColumn("date", to_date(col("date"))) df_with_year = df_cleaned.withColumn("year", year(col("date"))) # Aggregate data yearly_avg = df_with_year.groupBy("year").avg("avg_temp") # Show results yearly_avg.show()
