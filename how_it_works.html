Step 1: Data Loading

The process begins by loading weather data from a CSV file using PySpark:

# Read weather data from CSV df = spark.read.csv("weather_data.csv", header=True, schema=schema)

Step 2: Data Transformation

Data is cleaned and transformed for analysis:

# Convert date strings to DateType df_cleaned = df.withColumn("date", to_date(col("date"), "yyyy-MM-dd")) # Extract date components df_transformed = df_cleaned.withColumn("year", year(col("date")))

Step 3: Aggregation & Analysis

Various aggregations are computed to extract insights:

# Compute yearly metrics yearly_metrics = df.groupBy("year").agg( min("min_temp").alias("min_temp"), max("max_temp").alias("max_temp"), avg("avg_temp").alias("avg_temp") ) # Find extreme weather days hottest_days = df.orderBy(col("max_temp").desc()).limit(5)

Step 4: Results Generation

Finally, results are formatted and displayed:

# Show results yearly_metrics.show() hottest_days.show() # Generate insights print(f"Temperature trend: {temp_change:.2f}°C change")
